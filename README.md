# Melhor Oferta Imobiliária

Este repositório contém uma estrutura simples para um site estático a ser hospedado na Vercel.

## Como usar

1. Substitua `index.html` pelo seu próprio código HTML.
2. Conecte este repositório na sua conta Vercel (https://vercel.com/import/git).
3. Faça deploy automático ao fazer push no GitHub.

A configuração está pronta para sites estáticos. Se necessário, ajuste o arquivo `vercel.json`.
